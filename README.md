## Youtube Clone
Using React libraries, a fully responsive YouTube Clone was created with a stylish design that is compatible with any screen.


## Kütüphaneler
* https://tailwindcss.com/ (tailwind)
* react-icons
* react-router-dom
* react-player
* millify(sayı formatlama)
* moment(zaman formatlama)
* axios
